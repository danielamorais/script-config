# script-config
Script to generate SSH key, install and configure vim and zsh
```
$ git clone https://github.com/danielamorais/script-config.git
$ cd script-config/
$ chmod +x config-minimal.sh
$ ./config-minimal
```

## Plugins oh-my-zsh
* git
* pip
* python
* zsh-autosuggestions
* zsh-syntax-highlighting

## .vimrc 
* pathogen
